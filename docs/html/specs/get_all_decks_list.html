Get all decks list using a valid userId and valid filter

  • When the user invokes the API using the access token <file: /resources/access_tokens/valid_token.txt> and the following query parameters
    Query Parameter Query Value
    userId Osanda
    filter all
  • Then the status code for the request is "200"
  • And the JSON Path Assertions for the response should be equal to the following
    JSON Path Value
    $.[:1].title QE Test3
    $.[:1].subject.name SUB 1
    $.[:1].userId Osanda
